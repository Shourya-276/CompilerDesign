1. Install the bison and flex tool by running the following command:
`sudo apt install bison flex`

2. Download the 3 files attached below, namely `Makefile`, `bison.y`
and `flex.l`

3. Move the Downloaded files to a new directory named `ex4` by running
the following commands:
`mkdir ~/ex4`
`mv ~/Downloads/Makefile ~/ex4`
`mv ~/Downloads/bison.y ~/ex4`
`mv ~/Downloads/flex.l ~/ex4`

4. Build the executable:
`cd ~/ex4`
`make`

5. Execute a.out:
`./a.out`
